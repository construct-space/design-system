Getting Started

Welcome to Construct

Build, deploy, and manage your AI-powered workspaces with ease.

Quick Setup

To get started, install the CLI tool using your preferred package manager. Then run construct init in your project directory.

Configuration

Create a construct.yaml file at the root of your project. This file controls space types, agent behavior, and connected services.

# construct.yaml

name: my-project

spaces:

- type: code

- type: docs

Truncation

Text overflow patterns used in constrained UI elements.

Single-line truncate

This is a very long piece of text that should be truncated with an ellipsis when it overflows the container boundary

Two-line clamp

This is a longer piece of text that should be clamped to exactly two lines. When the text exceeds two lines it will be cut off with an ellipsis at the end of the second line. This demonstrates the multi-line truncation pattern.
